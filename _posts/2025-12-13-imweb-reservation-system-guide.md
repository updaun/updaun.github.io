## 1. 아임웹 예약 시스템 소개

### 1.1 왜 아임웹으로 예약 시스템을 만드는가?

예약 시스템은 현대 비즈니스에서 필수적인 도구입니다. 미용실, 병원, 상담센터, 학원, 스터디룸, 펜션 등 시간이나 공간을 기반으로 서비스를 제공하는 모든 비즈니스에서 고객이 편리하게 예약할 수 있는 시스템이 필요합니다.

전통적으로 예약 시스템을 구축하려면 개발자를 고용하거나 고가의 SaaS 솔루션을 구독해야 했습니다. 하지만 **아임웹(imweb)**을 사용하면 코딩 지식 없이도 전문적인 예약 시스템을 구축할 수 있습니다.

**아임웹 예약 시스템의 장점:**

✅ **노코드 구축** - 코딩 없이 드래그 앤 드롭으로 제작
✅ **빠른 구축** - 1~2일이면 완성 가능
✅ **반응형 디자인** - 모바일, 태블릿, PC 자동 대응
✅ **결제 연동** - 이니시스, KG이니시스, 토스페이먼츠 등 연동
✅ **예약 관리** - 실시간 예약 현황 확인 및 관리
✅ **알림 기능** - SMS, 카카오톡 알림 자동 발송
✅ **합리적 가격** - 월 16,500원부터 시작 (예약 기능 포함)
✅ **한국형 서비스** - 한국 결제 시스템 및 배송 시스템 완벽 지원

### 1.2 아임웹 예약 시스템 활용 사례

아임웹으로 구축할 수 있는 다양한 예약 시스템:

**1) 뷰티/헬스케어**
- 미용실 예약 (헤어, 네일, 메이크업)
- 피부관리실, 마사지샵
- 병원/한의원 진료 예약
- 필라테스, 요가 수업 예약

**2) 교육/문화**
- 학원 수업 예약
- 1:1 과외 예약
- 음악 레슨 (피아노, 기타, 보컬)
- 요리 클래스, 공방 체험

**3) 공간 대여**
- 스터디룸, 회의실 예약
- 파티룸, 연습실 대여
- 펜션, 게스트하우스 숙박 예약
- 스포츠 시설 (테니스장, 풋살장)

**4) 전문 서비스**
- 법률/세무 상담 예약
- 코칭/컨설팅 세션
- 심리 상담 예약
- 반려동물 미용/호텔

### 1.3 필요한 준비물

아임웹 예약 시스템 구축을 시작하기 전에 준비해야 할 것들:

**필수 준비물:**
- ✅ 아임웹 계정 (무료 체험 가능)
- ✅ 사업자등록번호 (결제 연동 시 필요)
- ✅ 예약 서비스 내용 및 가격 정리
- ✅ 운영 시간 및 휴무일 정보
- ✅ 대표 이미지 및 설명 자료

**선택 준비물:**
- 📱 카카오톡 알림톡 계정 (알림 발송용)
- 💳 PG사 계약 (이니시스, KG이니시스 등)
- 🏢 도메인 (imweb.me 무료 도메인도 가능)
- 📧 비즈니스 이메일 주소

## 2. 시작하기 - 기본 설정

### 2.1 아임웹 회원가입 및 사이트 생성

**1단계: 아임웹 회원가입**

```
1. https://imweb.me 접속
2. 우측 상단 [무료로 시작하기] 클릭
3. 이메일/비밀번호 입력 또는 소셜 로그인
4. 휴대폰 인증 완료
```

**2단계: 사이트 만들기**

회원가입 후 자동으로 사이트 생성 화면으로 이동합니다.

```
1. 사이트 이름 입력 (예: "강남헤어살롱 예약")
2. 사이트 주제 선택
   - 서비스/예약 선택
3. 템플릿 선택
   - 예약 전용 템플릿 추천: "Booking", "Reserve", "Appointment"
   - 빈 템플릿으로 시작도 가능
4. [사이트 만들기] 클릭
```

**3단계: 도메인 설정**

```
1. 관리자 페이지 → [사이트 설정] → [도메인]
2. 무료 도메인 설정
   - 원하는 주소 입력: yourbusiness.imweb.me
3. 또는 독립 도메인 연결
   - 별도 구매한 도메인 (예: yourbusiness.com) 연결
```

### 2.2 예약 기능 활성화

아임웹에서 예약 시스템을 사용하려면 쇼핑몰 기능을 활성화해야 합니다. 예약은 '예약 상품'으로 등록되기 때문입니다.

**쇼핑몰 기능 활성화:**

```
1. 관리자 페이지 좌측 메뉴에서 [쇼핑몰]이 보이는지 확인
2. 보이지 않으면:
   - [사이트 설정] → [쇼핑몰 설정] 접속
   - [쇼핑몰 사용 설정] ON
3. 기본 정보 입력:
   - 대표자명
   - 사업자등록번호 (없으면 나중에 입력 가능)
   - 통신판매업 신고번호 (있는 경우)
   - 사업장 주소
   - 고객센터 연락처
```

**예약 모듈 활성화:**

```
1. [쇼핑몰] → [상품 관리] → [상품 설정]
2. [예약 상품 사용] ON으로 변경
3. 예약 기본 설정:
   - 예약 가능 기간: 최대 90일 전부터 예약 가능
   - 예약 마감 시간: 예약일 1일 전까지
   - 동시 예약 건수 제한: 시간대별 1~10건
```

### 2.3 요금제 선택

아임웹은 여러 요금제를 제공하며, 예약 기능은 **라이트 플랜** 이상에서 사용 가능합니다.

**요금제 비교:**

| 기능 | 무료 | 라이트 | 프로 | 프리미엄 |
|------|------|--------|------|----------|
| 월 요금 | 0원 | 16,500원 | 27,500원 | 49,500원 |
| 예약 기능 | ❌ | ✅ | ✅ | ✅ |
| 상품 등록 | 10개 | 100개 | 1,000개 | 무제한 |
| 결제 연동 | ❌ | ✅ | ✅ | ✅ |
| SMS 알림 | ❌ | ✅ (100건) | ✅ (200건) | ✅ (500건) |
| 데이터 용량 | 100MB | 10GB | 50GB | 200GB |

**추천 요금제:**

- **소규모 예약 (1인 운영)**: 라이트 플랜
- **중규모 예약 (다수 스태프)**: 프로 플랜
- **대규모 예약 (체인점, 프랜차이즈)**: 프리미엄 플랜

**요금제 업그레이드:**

```
1. 관리자 페이지 → [내 정보] → [요금제 관리]
2. 원하는 플랜 선택
3. 결제 수단 등록 (신용카드, 계좌이체)
4. 즉시 업그레이드 완료
```

💡 **팁**: 14일 무료 체험이 가능하므로, 먼저 체험판으로 시스템을 구축한 후 만족하면 유료로 전환하세요.

### 2.4 기본 페이지 구성

예약 사이트에 필요한 기본 페이지를 구성합니다.

**필수 페이지:**

1. **홈페이지** (메인)
   - 서비스 소개
   - 예약 바로가기 버튼
   - 주요 서비스 이미지

2. **예약 페이지**
   - 예약 가능한 서비스 목록
   - 예약 달력/시간 선택

3. **서비스 소개**
   - 제공 서비스 상세 설명
   - 가격표
   - 시설 사진

4. **오시는 길**
   - 주소 및 지도
   - 교통편 안내
   - 주차 정보

5. **이용 안내**
   - 예약 방법 안내
   - 취소/환불 정책
   - FAQ

**페이지 만들기:**

```
1. 관리자 페이지 → [페이지 편집]
2. 좌측 상단 [+] 버튼 → [새 페이지 추가]
3. 페이지 정보 입력:
   - 페이지명: 예) "예약하기"
   - URL: /reservation
   - 메뉴 노출: ON
4. 템플릿 선택 또는 빈 페이지로 시작
5. [페이지 만들기] 클릭
```

이제 기본 설정이 완료되었습니다. 다음 섹션에서는 실제 예약 상품을 등록하는 방법을 알아보겠습니다.

## 3. 예약 상품 등록하기

### 3.1 예약 상품 생성

예약 서비스는 '상품'으로 등록됩니다. 각 서비스(예: 헤어컷, 펌, 염색)를 개별 상품으로 만들어야 합니다.

**상품 등록 절차:**

```
1. 관리자 페이지 → [쇼핑몰] → [상품 관리]
2. 우측 상단 [상품 등록] 버튼 클릭
3. 상품 유형 선택:
   - [예약 상품] 선택 (중요!)
4. 기본 정보 입력 시작
```

### 3.2 기본 정보 입력

**필수 입력 항목:**

**1) 상품명**
```
- 명확하고 간결하게 작성
- 좋은 예: "여성 헤어컷 (샴푸 포함)"
- 나쁜 예: "컷"
```

**2) 판매가**
```
- 서비스 가격 입력
- 예약금만 받는 경우: 예약금 금액 입력
- VAT 포함 여부 선택
```

**3) 상품 이미지**
```
- 권장 크기: 1000x1000px 이상
- 정사각형 비율 권장
- 서비스 결과물 사진 또는 시설 사진
- 최소 1장, 권장 3~5장
```

**4) 상품 설명**
```
서비스 상세 설명 작성:

[헤어컷 서비스]

✂️ 포함 내용
- 전문 스타일리스트 상담
- 샴푸 및 트리트먼트
- 스타일링 (드라이 포함)

⏱️ 소요 시간: 약 60분

💡 이런 분께 추천해요
- 새로운 스타일에 도전하고 싶으신 분
- 손상된 모발을 케어하고 싶으신 분
- 특별한 날을 앞두신 분

📋 유의사항
- 염색/펌과 동일 예약 불가
- 당일 취소 시 예약금 환불 불가
```

**5) 카테고리**
```
상품을 분류할 카테고리 선택:
- 헤어 → 컷, 펌, 염색
- 네일 → 젤네일, 케어, 아트
- 스터디룸 → 1인실, 2인실, 4인실
```

### 3.3 예약 옵션 설정

예약 상품의 핵심은 시간 설정입니다.

**예약 시간 설정:**

```
[상품 등록] 화면에서 아래로 스크롤

1. [예약 설정] 섹션 찾기

2. 예약 가능 요일 선택:
   ☑ 월요일
   ☑ 화요일
   ☑ 수요일
   ☑ 목요일
   ☑ 금요일
   ☑ 토요일
   ☐ 일요일 (휴무)

3. 예약 가능 시간 설정:
   - 시작 시간: 10:00
   - 종료 시간: 20:00
   - 시간 간격: 30분 단위

4. 소요 시간 설정:
   - 서비스 제공 시간: 60분
   - 준비 시간: 10분 (다음 고객 준비)
   
5. 동시 예약 가능 인원:
   - 1명 (1:1 서비스)
   - 또는 2~10명 (동시 수용 가능 인원)
```

**예약 기간 제한:**

```
1. 최소 예약 시간:
   - 1시간 전까지 예약 가능
   - 또는 1일 전까지 (추천)

2. 최대 예약 가능 기간:
   - 30일 후까지
   - 60일 후까지
   - 90일 후까지

3. 예약 가능 개수 제한:
   - 1인당 동시 예약: 최대 3개
```

### 3.4 추가 옵션 설정

서비스에 추가할 수 있는 옵션을 설정합니다.

**옵션 추가 예제 (헤어 서비스):**

```
1. [옵션 추가] 버튼 클릭

2. 옵션 1: 담당 스타일리스트
   - 옵션명: "스타일리스트 선택"
   - 선택 방식: 단일 선택
   - 옵션값:
     * 원장 김OO (추가금 +10,000원)
     * 디자이너 이OO (추가금 +5,000원)
     * 디자이너 박OO (추가금 +5,000원)
     * 상관없음 (추가금 없음)

3. 옵션 2: 추가 서비스
   - 옵션명: "추가 트리트먼트"
   - 선택 방식: 단일 선택
   - 옵션값:
     * 기본 트리트먼트 (포함)
     * 프리미엄 트리트먼트 (+20,000원)
     * 케라틴 트리트먼트 (+30,000원)
```

**스터디룸 예약 옵션 예제:**

```
옵션 1: 인원 수
- 1인 (기본 가격)
- 2인 (+5,000원)
- 3인 (+10,000원)
- 4인 (+15,000원)

옵션 2: 추가 시설
- 빔 프로젝터 (+10,000원)
- 화이트보드 (+5,000원)
- 간단한 다과 (+15,000원)
```

### 3.5 재고 및 판매 설정

```
1. 재고 관리:
   - [재고 관리 사용] OFF (예약 상품은 시간으로 관리)
   
2. 판매 상태:
   - [판매중] 선택
   - 일시적으로 예약 중단: [품절] 선택

3. 판매 기간:
   - 제한 없음 (계속 판매)
   - 또는 특정 기간만 판매 (이벤트성 서비스)

4. 회원 등급별 가격:
   - VIP 회원: 10% 할인
   - 일반 회원: 정상가
```

### 3.6 배송/픽업 설정 (예약에서는 사용 안 함)

예약 서비스는 배송이 필요 없으므로:

```
1. 배송 방법: [직접수령] 선택
2. 배송비: 0원
3. 기타 배송 옵션 모두 OFF
```

### 3.7 상품 저장 및 확인

```
1. 하단 [저장] 버튼 클릭
2. [쇼핑몰] → [상품 관리]에서 등록된 상품 확인
3. [미리보기] 버튼으로 고객 화면 확인

검토 사항:
✅ 상품명이 명확한가?
✅ 가격 정보가 정확한가?
✅ 이미지가 잘 보이는가?
✅ 예약 가능 시간이 올바른가?
✅ 옵션이 제대로 표시되는가?

## 4. 예약 시간 및 날짜 설정

### 4.1 예약 달력 설정

아임웹은 자동으로 예약 가능한 날짜와 시간을 달력 형태로 표시합니다.

**달력 표시 방식:**

```
1. [쇼핑몰] → [예약 설정] → [예약 달력 설정]

2. 달력 표시 형식:
   - 월간 달력 (한 달 전체 보기)
   - 주간 달력 (1주일 단위)
   - 일간 타임 슬롯 (시간대별 선택)

3. 예약 가능 표시:
   - 예약 가능: 초록색
   - 예약 마감: 회색
   - 휴무일: 빨간색
```

### 4.2 휴무일 및 특별 운영일 설정

**정기 휴무일:**

```
1. [쇼핑몰] → [예약 설정] → [휴무일 관리]
2. [정기 휴무 추가]
   - 매주 일요일
   - 매월 둘째, 넷째 월요일
3. [저장]
```

**임시 휴무일 (공휴일, 명절):**

```
1. [임시 휴무 추가] 클릭
2. 날짜 선택:
   - 2025-01-01 (신정)
   - 2025-02-09~02-12 (설날)
   - 2025-05-05 (어린이날)
3. 사유 입력: "명절 휴무"
4. [저장]
```

**특별 운영 시간:**

```
특정 날짜에만 다른 시간 운영:

예: 12월 31일 조기 마감
- 날짜: 2025-12-31
- 운영 시간: 10:00 ~ 15:00
- 사유: "연말 조기 마감"
```

### 4.3 시간대별 예약 설정

**복잡한 시간 설정이 필요한 경우:**

```
예: 점심시간 예약 불가

월~금요일:
- 오전 타임: 10:00 ~ 12:00
- 점심시간: 12:00 ~ 13:00 (예약 불가)
- 오후 타임: 13:00 ~ 18:00
- 저녁 타임: 18:00 ~ 20:00

토요일:
- 10:00 ~ 17:00 (점심시간 없음)
```

**설정 방법:**

```
1. 각 시간대별로 별도 상품 생성
   - 상품명: "헤어컷 (오전)"
   - 예약 시간: 10:00 ~ 12:00
   
   - 상품명: "헤어컷 (오후)"
   - 예약 시간: 13:00 ~ 18:00

2. 또는 예약 불가 시간 설정:
   - [예약 설정] → [예약 불가 시간]
   - 12:00 ~ 13:00 추가
```

### 4.4 예약 마감 시간 설정

```
[쇼핑몰] → [예약 설정] → [예약 마감 설정]

1. 당일 예약 마감:
   - 예약일 1시간 전까지 가능
   - 또는 예약일 3시간 전까지
   - 또는 당일 예약 불가

2. 사전 예약 마감:
   - 최소 1일 전까지 예약 필수
   - 최소 3일 전까지 예약 필수

추천: 
- 긴급 서비스: 1시간 전까지
- 일반 서비스: 3~6시간 전까지
- 준비가 필요한 서비스: 1일 전까지
```

## 6. 결제 시스템 연동

### 6.1 PG사 연동하기

**이니시스 연동 (가장 많이 사용):**

```
1. 이니시스 가맹점 신청
   - https://www.inicis.com
   - 사업자등록증 준비
   - 신청 후 1~3일 승인

2. 아임웹 연동:
   - [쇼핑몰] → [결제 설정]
   - [PG사 설정] → [이니시스]
   - MID 입력 (이니시스에서 발급)
   - 테스트 모드로 먼저 테스트
   - 실서비스 전환
```

### 6.2 결제 수단 설정

```
[쇼핑몰] → [결제 설정] → [결제 수단]

활성화할 결제 수단:
☑ 신용카드
☑ 실시간 계좌이체
☑ 가상계좌
☑ 휴대폰 결제
☑ 카카오페이
☑ 네이버페이
☐ 무통장 입금 (권장하지 않음)

예약금만 받는 경우:
- "예약금 결제" 명시
- 나머지 금액은 현장 결제 안내
```

### 6.3 결제 정책 설정

```
[결제 설정] → [정책 관리]

1. 최소 주문 금액: 0원 (제한 없음)
2. 결제 완료 후 처리:
   - 주문 상태: "결제완료" → "예약확정"
   - 자동 문자 발송
3. 부가세 처리:
   - 부가세 별도 OR 포함 선택
4. 현금영수증:
   - 자동 발급 설정
```

### 6.4 예약금 vs 전액 결제

**예약금 방식:**
```
장점:
- 노쇼 방지
- 가볍게 예약 가능
- 취소 시 부담 적음

설정:
- 상품 가격: 예약금 금액 (10,000원)
- 상품 설명에 명시: "예약금이며, 나머지 40,000원은 현장 결제"
```

**전액 결제 방식:**
```
장점:
- 당일 결제 불필요
- 매출 확실히 확보
- 정산 간편

설정:
- 상품 가격: 서비스 전체 금액
- 취소 정책 명확히 안내
```

- 취소 발생 시 자동으로 대기 순번 고객에게 알림
- "대기 예약 하시겠습니까?" 옵션 제공
- 대기 순번 안내

설정:
[예약 설정] → [대기 예약] ON
```

### 8.5 다중 지점 관리

```
여러 매장이 있는 경우:

1. 매장별 카테고리 생성:
   - 강남점
   - 홍대점
   - 판교점

2. 각 지점별로 상품 등록:
   - "헤어컷 (강남점)"
   - "헤어컷 (홍대점)"

3. 지점 선택 옵션 추가:
   - 예약 시 지점 먼저 선택
   - 해당 지점 서비스만 표시
```

## 10. 문제 해결 및 팁

### 10.1 자주 발생하는 문제

**Q1: 예약 달력이 표시되지 않아요**
```
해결방법:
1. 상품 유형이 "예약 상품"인지 확인
2. 예약 가능 요일과 시간이 설정되어 있는지 확인
3. 캐시 삭제 후 새로고침
4. 다른 브라우저에서 테스트
```

**Q2: 결제가 안 돼요**
```
해결방법:
1. PG사 설정이 완료되었는지 확인
2. 테스트 모드가 아닌 실서비스 모드인지 확인
3. MID 입력이 정확한지 확인
4. PG사 가맹점 승인이 완료되었는지 확인
```

**Q3: SMS가 발송되지 않아요**
```
해결방법:
1. SMS 크레딧이 남아있는지 확인
2. 발신번호 등록이 되어있는지 확인
3. 알림 설정이 ON인지 확인
4. 수신번호 형식이 올바른지 확인 (010-0000-0000)
```

**Q4: 모바일에서 레이아웃이 깨져요**
```
해결방법:
1. [디자인 설정] → [반응형] 확인
2. 이미지 크기가 너무 크지 않은지 확인
3. 위젯 너비 설정을 "자동"으로 변경
4. 모바일 프리뷰로 테스트
```

### 10.2 운영 팁

**팁 1: 노쇼 방지하기**
```
✅ 예약금 받기 (최소 10,000원)
✅ 예약 1일 전 SMS 리마인더
✅ 예약 당일 아침 재확인 문자
✅ 노쇼 3회 시 블랙리스트 등록
```

**팁 2: 예약률 높이기**
```
✅ 상품 이미지를 고퀄리티로
✅ 상세 설명 충실히 작성
✅ 리뷰 시스템 활용
✅ 첫 예약 할인 쿠폰 제공
✅ SNS 광고 연동
```

**팁 3: 효율적인 시간 관리**
```
✅ 인기 시간대 파악 (통계 활용)
✅ 바쁜 시간은 가격 올리기 (성수기 요금)
✅ 한가한 시간은 할인 프로모션
✅ 준비 시간 충분히 설정 (10~15분)
```

**팁 4: 고객 재방문 유도**
```
✅ 회원 등급별 할인
✅ 누적 포인트 제도
✅ 재방문 쿠폰 자동 발급
✅ 생일 쿠폰 발송
✅ 정기 예약 할인
```

### 10.3 SEO 최적화

```
예약 사이트 검색 노출 높이기:

1. 메타 태그 설정:
   - [사이트 설정] → [SEO 설정]
   - 제목: "서울 강남 헤어샵 예약 | OO살롱"
   - 설명: "강남역 도보 3분, 전문 스타일리스트, 온라인 예약 시 10% 할인"
   - 키워드: "강남 헤어샵, 헤어샵 예약, 강남역 미용실"

2. 상품 설명 최적화:
   - 지역명 포함
   - 서비스 상세 설명
   - 이미지 alt 태그 설정

3. 네이버 플레이스 연동:
   - 아임웹 사이트 주소 등록
   - 예약 링크 연결

4. 구글 마이 비즈니스:
   - 사이트 주소 등록
   - 예약 버튼 추가
```

### 10.4 보안 및 개인정보 보호

```
개인정보 처리방침:

1. 필수 페이지 생성:
   - 개인정보 처리방침
   - 이용약관
   - 환불 정책

2. [사이트 설정] → [약관 설정]
   - 각 약관 등록
   - 예약 시 동의 필수로 설정

3. SSL 인증서:
   - 아임웹 자동 제공 (https)
   - 독립 도메인도 무료 SSL

4. 고객 정보 관리:
   - 주기적으로 불필요한 정보 삭제
   - 접근 권한 관리
   - 백업 정기적으로 실행
```
